🪨📄✂️ Jokenpô em Python

Um jogo clássico de Pedra, Papel e Tesoura feito em Python, executado no terminal.
Projeto simples com foco em prática de lógica, estruturas condicionais e organização de código.

🚀 Funcionalidades
Escolha do jogador via input (menu numérico)
Escolha aleatória do computador
Sistema de regras do jogo
Validação de entrada
Loop contínuo para múltiplas partidas
🧠 Conceitos aplicados
while (loop infinito controlado)
if/elif/else (condições)
random.choice() (aleatoriedade)
Dicionários (dict) para mapear escolhas
Estrutura de regras otimizada
📂 Estrutura do código
opcoes → lista com escolhas possíveis
traducao → converte input numérico em texto
regras → define quem vence de quem
▶️ Como executar
Certifique-se de ter o Python instalado (3.x)
Clone o repositório:
git clone https://github.com/seu-usuario/jokenpo-python.git

Acesse a pasta:
cd jokenpo-python

Execute o arquivo:
python jokenpo.py

🎮 Como jogar

Digite uma das opções:

[1] Pedra
[2] Papel
[3] Tesoura


O sistema mostrará o resultado da rodada.

📌 Exemplo de execução
-------------------- JOKENPO --------------------
Qual opção você deseja?
[1] Pedra
[2] Papel
[3] Tesoura
> 1

O Player escolheu: pedra
O PC escolheu: tesoura

Você venceu!
